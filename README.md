# Jogo-da-Velha
Jogo da Velha em C

-- Versão atual --
* Presente:
- Detecção do jogador vencedor ou velha (sem influência no código).
- Detecção de vitórias nas diagonais.
- Função de pause.
- Uso de ponteiros

* Em progresso:
- Detecção de vitorias nas linhas e colunas.
- Cores no terminal.
- Tratamento de erros de entrada.
- Opção de jogo contra máquina.
